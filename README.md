# 급여 계산 시스템 요구사항


## 구현해야 하는 기능
1. 일반 직원의 급여 계산하기 : 기본급 - 기본급 * 소득세율
2. 아르바이트 직원의 급여 계산하기 : 계산 방법은 일반 직원과 동일, 기본급은 일한시간 * 시급으로 계산
3. 모든 일반 직원의 급여 총합 계산하기

## 입출력 요구사항
숫자는 예시임
1. 실행할 기능 선택
```
급여 계산 시스템

실행할 기능을 선택하세요
1. 급여 계산
2. 모든 일반 직원의 급여 총합 계산

실행할 기능 입력(숫자로) : 
```

2. 급여 계산

```
직원의 아이디를 입력하세요 : 1234
소득세율을 입력하세요(% 단위) : 50

소득 계산 결과
이름: 민휘, 급여: 4,350,000원
```

3. 모든 일반 직원의 급여 총합 계산
```
모든 일반 직원의 급여 총합을 계산하고 있습니다 . . .

일반 직원의 급여 총합 계산 결과
인원: 23명, 급여: 100,150,000원
```
